<h1>git hero</h1>

Test your git command knowledge in this fast-paced typing game. 
Git commands fall from the sky — type them correctly before they hit the bottom. 
Miss 3 commands and it's game over. 
Reach 200 points to win.

**Controls:**
Type the falling command and press **ENTER** to submit
